# Cardiovascular Disease Prediction

## Project Overview
This project predicts cardiovascular disease using Machine Learning algorithms.

## Dataset
- cardio_train.csv

## Machine Learning Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Accuracy Comparison

| Algorithm | Accuracy |
|-----------|----------|
| Logistic Regression | 72.78% |
| Decision Tree | 63.04% |
| Random Forest | 72.23% |
| KNN | 68.76% |
| SVM | 73.50% |

## Best Model
Support Vector Machine (SVM)

Accuracy: **73.50%**
